## Notes
- **skip-back-15/30** share the `rewind-back` glyph; **skip-forward-15/30** share `rewind-forward`. The 15/30 numerals are intended as a UI text overlay.
- **loop-a / loop-b** share the `repeat-one` glyph; overlay the A/B letter in the UI. **clear-loop** uses the minimalistic repeat-one variant.
- **play-along** uses `keyboard` (Solar has no piano-keyboard glyph).
- **connection** uses `bluetooth-square` (matches the BLE-MIDI link).
- **record** is a filled circle (`record`); pairs with **stop** (filled square).
